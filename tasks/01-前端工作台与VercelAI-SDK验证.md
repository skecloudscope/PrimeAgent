# 01 前端工作台与 Vercel AI SDK 验证

## 研究目标

验证 Next.js + Vercel AI SDK 是否适合承载跨境电商 Agent 工作台，包括流式对话、工具调用状态、审批状态和 FastAPI 后端对接。

## 业务场景

用户在工作台中选择 Shopify 商品，触发 Listing 优化 Agent，前端展示 Agent 流式输出、工具调用、待审批动作和最终写回结果。

## 需要阅读的源码

- Vercel AI SDK core package。
- Vercel AI SDK React hooks。
- Vercel AI SDK streaming protocol。
- Vercel AI SDK tool calling examples。
- Vercel AI SDK structured output examples。
