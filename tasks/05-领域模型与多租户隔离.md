# 05 数据模型与多租户隔离

## 研究目标

设计跨境电商 Agent 平台的核心业务对象和多租户隔离模型。

## 业务场景

一个用户可以加入多个租户，一个租户可以管理多个 workspace 和 shop。每个 Agent、Workflow、Connector、Memory、Approval、Audit 都必须能追溯到租户和店铺。

## 需要阅读的源码

- Agno DB schemas。
- Agno session / memory / approval / trace 存储模型。
- Nango connection 数据模型。
- Clerk user / organization token 结构。
