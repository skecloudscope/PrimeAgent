# 06 权限、工具网关与审批验证

## 研究目标

验证工具调用如何经过业务权限网关，并让所有外部写操作进入审批。

## 业务场景

Listing 优化 Agent 想写回 Shopify 商品时，系统检查 user、tenant、shop、agent、tool 权限，创建审批，审批通过后继续执行写操作。
